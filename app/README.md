## Stack
FrontEnd:
- React
- Apollo Client
- GraphQL
- Bootstrap

BackEnd:
- NodeJS
- Express
- GraphQL
- Apollo Server
- MongoDB

## Start

Front:
```
cd app
npm install
npm start
```

Back:
```
cd app/backend
npm install
npm start
```

React runs on : http://localhost:3000

Backend runs on : http://localhost:5000
